About This Tool

PDF to Audio Converter is a lightweight web tool built to help users upload a PDF, extract readable text, and turn that content into downloadable MP3 audio.

The project is designed as a separate AdSense-friendly utility page with a clear content structure, mobile-friendly layout, and a practical use case for students, professionals, and accessibility-focused users.

On the backend, the app uses FastAPI for uploads and processing, PyMuPDF for PDF text extraction, OCR fallback for scanned documents, and an internal text-to-speech API for audio generation.

The goal is simple: make everyday PDF listening fast, usable, and easy to access from the browser.
